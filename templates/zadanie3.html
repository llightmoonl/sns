Задание 3. Классификация рукописных цифр при помощи СНС

Цель задания: опробовать модель сверточной нейронной сети, построенной с использованием датасета mnist, для распознавания рукописных цифр.

Для начала еще раз ознакомимся с архитектурой модели, которую мы выстроили в прошлом задании:

model = Sequential([

Conv2D(32, (3,3), activation="relu", input_shape=(28,28,1)),

MaxPooling2D((2,2)),

Conv2D(64, (3,3), activation="relu"),

MaxPooling2D((2,2)),

Flatten(),

Dense(64, activation="relu"),

Dense(10, activation="softmax")

])

Вспомним, что из себя представляет каждый слой СНС. Для начала рассмотрим один из важных слоев – Conv2D. Он отвечает за процесс свертки. Принцип работы – скольжение по входной карте признаков с целью умножения значений матрицы на соответствующие веса, формирующиеся в ядре свертки. Получившиеся значения откладываются в соответствующие места на выходной карте признаков. На рисунке с визуализацией данного процесса можно увидеть, как фиолетовая матрица умножается на матрицу ядра свертки и получившееся значение (фиолетовый ноль) откладывается в выходной карте. Данный процесс повторяется со сдвигом на один шаг, пока все матрицы не умножатся на ядро свертки:

Операция свертки - Conv2D

Далее рассмотрим слой MaxPooling2D. Он отвечает за процесс пулинга. Принцип работы – уменьшение карты признаков, которая получается в процессе свертки. Уменьшение происходит за счет выбора максимального значения в каждой матрице и откладывание его на соответствующем месте в выходной карте признаков. На рисунке, визуализирующем данный процесс, можно заметить, что из зеленой матрицы с помощью процесса пулинга была выбрана максимальная цифра 2. Градация цвета от светлого к темному помогает нам увидеть цифры матрицы в порядке возрастания. Точно также происходит в оранжевой, фиолетовой и розовой матрицах:

Операция пулинга - MaxPooling2D

Слой Flatten отвечает за преобразование многомерных данных в одномерные. К моменту работы данного слоя у нас сформировано 64 массива, состоящих из 25 элементов. Слой Flatten преобразует данные массивы в одномерный массив, состоящий из 64*25 = 1600 элементов.

Операция выравнивания - Flatten

Одномерный массив попадет в полносвязный слой Dense. Данный слой необходим для первичной классификации ключевых признаков, которые передаются во второй полносвязный слой Dense. И именно в нем будет происходить окончательная классификации цифры, изображенной на входной картинке.

Теперь нам нужно вспомнить, как же работают функции активации в слоях сверточной нейронной сети и что они представляют из себя.

Функции активации – это определенные математические функции, которые принимают на вход значение и преобразуют его в выходное. Они применяются к каждому элементу данных, чтобы ввести в модель нелинейность. В нашей модели сверточной нейронной сети используются такие функции активации, как ReLU и Softmax.

Функция активации ReLU пропускает положительные входные значения без изменений, а отрицательные значения преобразует в ноль. Данная функция нужна, чтобы избежать отрицательных значений. График функции ReLU можно увидеть ниже:

Функция активации ReLU

Функция активации Softmax преобразует выходные значения нейронной сети в вероятности. Данная функция обычно используется в последнем полносвязном слое для окончательной классификации, например, рукописных цифр, животных, одежды или других объектов. График функции Softmax:
